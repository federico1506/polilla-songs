# Polilla - Banda Bahiense

Landing page oficial de la banda **Polilla**, una banda bahiense que fusiona diferentes estilos musicales para crear una experiencia única.

## 🎵 Integrantes

- **Bruno** - Baterista
- **Rodrigo** - Cantante
- **Federico** - Pianista
- **Juan Pablo** - Bajista
- **Bautista** - Guitarrista

## 🎼 Canciones

La banda cuenta con un repertorio variado que incluye:

- **Te encontraré** - Una canción sobre la búsqueda y el amor
- **Polilla** - Explorando el efecto polilla y las alas rotas
- **DES-Encontrar** - Un viaje cósmico y estelar
- **Nexo** - El dolor en el núcleo nuclear
- **El eco de tu voz** - Recuerdos y desvanecimientos
- **Flores rojas** - Psicodelia y fragilidad etérea
- **Abismos** - Todos los abismos son iguales

## 🚀 Tecnologías

- **React 18** - Framework de JavaScript
- **TypeScript** - Tipado estático
- **Vite** - Build tool y dev server
- **CSS Variables** - Sistema de colores personalizable
- **Responsive Design** - Optimizado para todos los dispositivos

## 🎨 Diseño

La landing page presenta un diseño moderno y elegante con:

- Tema oscuro con acentos dorados
- Animaciones suaves y transiciones
- Modal interactivo para ver las letras completas
- Diseño responsive para móviles y desktop
- Efectos de hover y animaciones de entrada

## 📦 Instalación

```bash
# Clonar el repositorio
git clone [url-del-repositorio]

# Entrar al directorio
cd polilla-songs

# Instalar dependencias
pnpm install

# Ejecutar en modo desarrollo
pnpm run dev

# Construir para producción
pnpm run build
```
